# Données de départ

Le fichier [`higher_education_sample.csv`](./higher_education_sample.csv) est un **jeu de données pédagogique simplifié**.

Il contient maintenant un **panel élargi d'établissements marocains** afin de rendre l'analyse plus riche :

- universités publiques
- écoles d'ingénieurs
- instituts publics spécialisés
- établissements situés dans plusieurs villes

Il sert à lancer rapidement le TP si vous ne souhaitez pas perdre de temps à chercher un jeu de données externe pendant la séance.

## Usage conseillé

- utilisez-le comme support d'analyse si vous souhaitez rester sur un cadre commun
- ou remplacez-le par un jeu de données réel trouvé sur un portail Open Data

## Important

Ce fichier n'est **pas** un jeu de données liées. Il est volontairement tabulaire et imparfait afin que vous puissiez analyser :

- ce qu'il exprime déjà
- ce qu'il ne permet pas encore de faire
- quels liens externes seraient utiles

## Idées d'entités à identifier

À partir de ce fichier, vous pouvez par exemple identifier :

- des établissements
- des villes
- des pays
- des domaines de formation
- des sites web institutionnels
- des portails de données
